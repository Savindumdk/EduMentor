## Technology Stack

### Phase 1 (Current):
- **Python 3.8+**: Core language
- **Experta**: Rule-based expert system
- **Streamlit**: Web UI framework

### Phase 2 (Planned):
- **SPADE** or **Mesa**: Multi-agent framework
- **Message Queue**: Inter-agent communication

### Phase 3 (Planned):
- **OpenAI GPT** or **Anthropic Claude**: LLM provider
- **LangChain**: LLM orchestration
- **Vector DB**: Enhanced retrieval (optional)
- **Translation API**: Multilingual support

## Design Patterns

1. **Separation of Concerns**: UI, Logic, and Data are separate
2. **Rule-Based Reasoning**: Experta forward chaining
3. **Modular Design**: Easy to extend and modify
4. **Future-Proof Architecture**: Hooks for MAS and LLM
5. **Progressive Enhancement**: Build on existing foundation
